# AirPoint
A mouse that is powered by an ESP32 and a MPU6050 along with two push buttons to control the computer's cursor by just pointing it!

# Why I Made This:
When I saw one of those Smart LG TV remotes that let you control a cursor on the TV, I wondered if I could do that, but on a computer. Feeling inspired, I learned that it was possible! I decided to make a simple one on a breadboard using three pushbuttons that will control the left and right clicks on the computer and also one that lets you scroll in any direction which is pretty unique (in my opinion)!
